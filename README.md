# Gemini 多账号自动生图工具

自动使用多个 Google 账号在 Gemini 网页端生成图片并下载的桌面工具，现已全面升级为 Web UI 控制台模式，并支持 Excel 批量复杂任务导入！

## ✨ 核心亮点

- **Web UI 任务队列** — 直接在网页输入提示词和目标张数，支持断点续传。
- **📊 强大的 Excel 批量导入** — 支持通过 Excel 导入任务，自动将大任务拆分为单张小任务。支持配置多列特征词（无限列数），每次生成均**随机抓取组合**，极大提高生图多样性！
- **📂 独立的自定义导出路径** — 支持在 Excel 中为不同行设定独立的图片保存路径（如为空则回退使用全局目录），方便按品类整理图片。
- **⚡ JS 剪贴板光速输入** — 底层采用 JS 剪贴板直贴技术，完美保留换行符且瞬间完成超长 Prompt 输入，彻底告别打字慢导致的漏发和截断。
- **智能“抢单”机制** — 多窗口共享中央任务池，谁先跑完谁接下一单，算力绝不闲置。
- **全自动防断连** — 额度用完自动换号（轮换），采用全局线程锁防死锁，运行异常自动重启对话，保证连续无缝作业。
- **无头不无脑** — 初次使用新账号需手动扫码/密码介入登录，登录完系统接管，真正做到安全第一。

## 🚀 启动方式

### 方式一：快捷方式启动（推荐 Windows 用户）
如果在桌面上生成了 `启动跑图机器人.bat` 快捷方式，只需双击它，程序会自动在后台运行，并直接打开浏览器控制台。

### 方式二：终端命令行启动
打开终端 (Terminal 或 cmd)，输入以下命令：
```cmd
cd 保存路径
python main.py
```

## 🛠 使用流程
1. 启动工具后，浏览器会自动打开 `http://127.0.0.1:5050`。
2. 配置**账号存放目录**和**全局默认输出目录**（只需配置一次）。
3. 设定**并行窗口数量**和**账号轮换**。
   - *提示：如果需要用到账号轮换（挂机多号），请在开始前为生成的每个新窗口手动登录 Google 账号。*
4. **添加任务**：
   - 方式 A：在右侧手动输入提示词和张数。
   - 方式 B：上传符合格式的 `.xlsx` 表格（B列总提示词、C列张数、D列保存路径、E列往后为需随机组合的特征词）。
5. 点击 **【1️⃣ 打开窗口准备登录】**。
6. 在弹出的 Chrome 窗口中，**手动完成 Google 账号的登录**（直到看到 Gemini 的输入框）。
7. 点击界面上的 **【2️⃣ 登录完成，开始跑图】**。
8. 脚本将全面接管浏览器，自动从任务队列里领任务、随机组合特征、跑图并按指定路径分发保存。

## 📁 目录结构
```
gemini-auto-gen/
├── main.py                    # 程序入口
├── web/                       # Web 服务端和前端 UI
├── core/
│   ├── browser_worker.py      # 浏览器工作核心逻辑（集成多线程锁）
│   ├── excel_parser.py        # Excel 批量解析与无限列随机组合引擎
│   ├── gemini_automation.py   # Gemini 网页自动化控制 (JS 剪贴板注入、精准找图)
│   ├── task_queue.py          # 中央任务池控制器
│   ├── account_manager.py     # 多账号隔离配置管理器
│   └── image_downloader.py    # 图片处理与保存
├── prompts.json               # 任务数据存储文件（自动生成）
├── config.json                # 配置存储文件（自动生成）
└── logs/                      # 运行日志
```

## ⚠️ 注意事项
- **黑色的终端窗口（Terminal）在运行期间绝对不能关闭**。这是工具的服务端后台，一旦关闭网页就无法工作了。
- 采用账号轮换机制时，对于每个首次生成的“隔离账号文件夹”（如 `window_1_account_02`），您需要手动登入一次 Google。
- 本地生成的 Chrome 账号配置文件如果占用空间过大，可以定期清理不再使用的账号数据。
